## Laravel Sneat

Laravel Starter with Sneat Admin Dashboard and some packages such as auth, permission and users management.

## Requirements
- Laravel (10.37.1)
- PHP (8.2.10)
- MySQL (8.0.30)

## Packages
- Breeze
- Spatie
- Socialite
- Php-Flasher
- Yajra Datatables

## Installation

- composer install
- npm install
- npm run dev or npm run build
- cp .env.example .env
- php artisan key:generate
- php artisan storage:link
