## Laravel Sneat

Laravel Starter with Sneat Admin Dashboard.

## Requirements
- Laravel (10.29.0)
- PHP (8.2.10)
- MySQL (8.0.30)

## Packages
- Breeze
- Spatie
- Socialite

## Installation

- composer install
- cp .env.example .env
- php artisan key:generate
- php artisan storage:link
